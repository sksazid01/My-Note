## Find out if the character belongs to the capital or small letter range.

* wrong:   if ((s[i] <= 'z' and s[i] >= 'A'))
* correct:  if ((s[i] <= 'Z' and s[i] >= 'A') or (s[i] <= 'z' and s[i] >= 'a'))
