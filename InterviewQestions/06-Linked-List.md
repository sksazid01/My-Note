## Create Data Structure uniqueQueue
**Create a data structure uniqueQueue where you can get the kth last element of the queue by using get(k) method and push n if it is not in the queue using push(n) method.**

Example:
```
push(11)
push(22)
push(33)
get(1) # returns 33
push(22)
get(2) # returns 22
```
